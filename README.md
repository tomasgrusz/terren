<h2>â–¶ï¸ Running Locally</h2>

1. Clone the repository
```bash
git clone https://github.com/tomasgrusz/terren.git
```

2. Navigate to project folder
```bash
cd terren
```

3. Install all dependencies
```bash
npm install
# or
yarn install
# or
bun install
```

4. Run the development environment

```bash
npm run dev

bun dev
```
